HTML:

Hyper text Markup Language.

Tag =>

syntax:

<keyword>

<html>

Tags are two types:

1) container tag
	start tag => content => End tag
	<html> your content </html>
	<p> your content </p>
2) empty tag

	<img>
	<br>
	<hr>


Prerequisites:

	1) Computer Basic knowledge
	2) Editor: Notepad, Notepad++, Sublime Text, Atom
	3) Browser: Mozila Firefox, Google Chrome

HTML Element:

<StartTag> content </EndTag> 

1) Single Element
		<p>This is a paragraph</p>
2) Nested Element.
		<p>This is <b>demo</b> paragraph</p>
		

HTML Attributes:

	1) Attributes are used for adding additional information to a HTML element.
	2) syntax: attributes_name = "attributes_value"
	3) Attributes should write in start tag
	
Example:

	<p attributes_name="value">This is example for attributes.</p>
	
HTML Style Attributes:

	syntax:
	
	attributes_name = "attributes_value"
	
	style = "property_name:property_value; property_name:property_value;"
